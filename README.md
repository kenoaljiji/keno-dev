## MERN stack Cos-ecommerce app

### how to run app

### Add on root .env file with variable

<ul>
    <li>NODE_ENV</li>
    <li>PORT</li>
    <li>MONGO_URI</li>
    <li>JWT_SECRET</li>
</ul>

#### npm install on root

#### navigate on frontend folder

#### npm install on frontend

#### npm run dev on root
